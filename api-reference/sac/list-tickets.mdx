---

## Descrição

Retorna uma lista paginada de tickets de suporte com opções de filtragem. Requer autenticação.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token JWT para autenticação
</ParamField>

## Query Parameters

<ParamField query="status" type="string" default="open">
  Filtrar por status do ticket
  
  Valores aceitos:
  - `all` - Todos os status
  - `open` - Tickets abertos
  - `in_progress` - Em atendimento
  - `resolved` - Resolvidos
  - `closed` - Fechados
</ParamField>

<ParamField query="main" type="string">
  Filtrar por categoria principal
</ParamField>

<ParamField query="issue" type="string">
  Filtrar por assunto
</ParamField>

<ParamField query="newsletter" type="string">
  Filtrar por newsletter
</ParamField>

<ParamField query="email" type="string">
  Filtrar por email do solicitante (busca parcial, case-insensitive)

  Exemplo: `email=amanda` retorna todos os tickets cujo email contém "amanda"
</ParamField>

<ParamField query="ticketNumber" type="number">
  Filtrar por número (ID) do ticket
</ParamField>

<ParamField query="tagId" type="number">
  Filtrar por ID da tag
</ParamField>

<ParamField query="startDate" type="string">
  Data inicial para filtro por período de criação (formato `YYYY-MM-DD`)
</ParamField>

<ParamField query="endDate" type="string">
  Data final para filtro por período de criação (formato `YYYY-MM-DD`)
</ParamField>

<ParamField query="page" type="number" default="1">
  Número da página
</ParamField>

<ParamField query="limit" type="number" default="10">
  Número de itens por página (máximo 100)
</ParamField>

<ParamField query="sort" type="string" default="desc">
  Ordem de classificação

  Valores aceitos:
  - `asc` - Crescente
  - `desc` - Decrescente
</ParamField>

<ParamField query="sortField" type="string" default="createdAt">
  Campo para ordenação

  Valores aceitos:
  - `createdAt` - Data de criação
  - `updatedAt` - Data de atualização
  - `email` - Email do solicitante
</ParamField>

<RequestExample>

```bash Básico
curl -X GET "https://worker.thenewscc.com/support" \
  -H "Authorization: Bearer seu_token_jwt"
```

```bash Com Filtros
curl -X GET "https://worker.thenewscc.com/support?status=open&main=Técnico&page=1&limit=20&sort=desc" \
  -H "Authorization: Bearer seu_token_jwt"
```

```bash Busca Parcial por Email
curl -X GET "https://worker.thenewscc.com/support?email=amanda&sortField=email&sort=asc" \
  -H "Authorization: Bearer seu_token_jwt"
```

```bash Filtrar por Período
curl -X GET "https://worker.thenewscc.com/support?startDate=2026-01-01&endDate=2026-01-31" \
  -H "Authorization: Bearer seu_token_jwt"
```

```bash Buscar por Número do Ticket
curl -X GET "https://worker.thenewscc.com/support?ticketNumber=12345" \
  -H "Authorization: Bearer seu_token_jwt"
```
