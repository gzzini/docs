---

## Descrição

Sistema de templates de respostas pré-definidas para agilizar o atendimento. Permite criar, editar, categorizar e reutilizar mensagens padrão.

## Endpoints Disponíveis

### Listar Respostas Rápidas

`GET /support/quick-replies`

#### Query Parameters

<ParamField query="categoria" type="string">
  Filtrar por categoria
</ParamField>

<ParamField query="busca" type="string">
  Buscar no título ou conteúdo
</ParamField>

<ParamField query="page" type="number" default="1">
  Número da página
</ParamField>

<ParamField query="limit" type="number" default="10">
  Itens por página
</ParamField>

### Criar Resposta Rápida

`POST /support/quick-replies`

#### Body Parameters

<ParamField body="titulo" type="string" required>
  Título da resposta rápida
</ParamField>

<ParamField body="conteudo" type="string" required>
  Conteúdo do template
</ParamField>

<ParamField body="categoria" type="string" required>
  Categoria da resposta
  
  Valores comuns:
  - `saudacao` - Mensagens de boas-vindas
  - `despedida` - Mensagens de encerramento
  - `tecnico` - Respostas técnicas
  - `financeiro` - Respostas financeiras
  - `geral` - Respostas gerais
</ParamField>

<ParamField body="tags" type="array">
  Tags para facilitar busca
</ParamField>

### Usar Resposta Rápida

`POST /support/quick-replies/{quickReplyId}/usar`

Incrementa o contador de uso e retorna o conteúdo formatado.

<RequestExample>

```bash Listar
curl -X GET "https://sac-waffle-v2.theo-110.workers.dev/support/quick-replies?categoria=tecnico" \
  -H "Authorization: Bearer seu_token_jwt"
```

```bash Criar
curl -X POST "https://sac-waffle-v2.theo-110.workers.dev/support/quick-replies" \
  -H "Authorization: Bearer seu_token_jwt" \
  -H "Content-Type: application/json" \
  -d '{
    "titulo": "Problema de Login",
    "conteudo": "Olá! Lamentamos o problema com seu login. Por favor, tente limpar o cache do navegador e os cookies. Se o problema persistir, clique em 'Esqueci minha senha' para redefini-la.",
    "categoria": "tecnico",
    "tags": ["login", "senha", "acesso"]
  }'
```

```bash Usar
curl -X POST "https://sac-waffle-v2.theo-110.workers.dev/support/quick-replies/123/usar" \
  -H "Authorization: Bearer seu_token_jwt"
```
