---
title: "Performance de Agentes"
api: "GET https://worker.thenewscc.com/support/relatorios/agentes"
description: "Retorna métricas de performance por agente"
---

## Descrição

Retorna métricas detalhadas de performance para cada agente que respondeu tickets no período, incluindo quantidade de tickets, tempos de resposta e resolução, e taxa de resolução. Este endpoint utiliza cache KV com TTL de 2 horas.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token JWT para autenticação
</ParamField>

## Query Parameters

<ParamField query="data_inicio" type="string">
  Data de início do período (formato: YYYY-MM-DD). Padrão: 30 dias atrás
</ParamField>

<ParamField query="data_fim" type="string">
  Data de fim do período (formato: YYYY-MM-DD). Padrão: hoje
</ParamField>

<ParamField query="main" type="string">
  Filtrar por categoria principal
</ParamField>

<ParamField query="newsletter" type="string">
  Filtrar por newsletter relacionada
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Dados de performance">
    <ResponseField name="agentes" type="array">
      <Expandable title="Estrutura de cada agente">
        <ResponseField name="agente" type="string">
          Email do agente
        </ResponseField>

        <ResponseField name="ticketsRespondidos" type="number">
          Total de tickets em que o agente respondeu
        </ResponseField>

        <ResponseField name="ticketsResolvidos" type="number">
          Total de tickets resolvidos pelo agente
        </ResponseField>

        <ResponseField name="tempoMedioResposta" type="string">
          Tempo médio de resposta formatado
        </ResponseField>

        <ResponseField name="tempoMedioResolucao" type="string">
          Tempo médio de resolução formatado
        </ResponseField>

        <ResponseField name="taxaResolucao" type="number">
          Taxa de resolução em percentual
        </ResponseField>

        <ResponseField name="satisfacao" type="number">
          Nota de satisfação (se disponível)
        </ResponseField>

        <ResponseField name="mensagensEnviadas" type="number">
          Total de mensagens enviadas pelo agente
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="periodo" type="object">
      Período analisado
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="cache" type="object">
  Informações de cache (hit, age, expiresIn)
</ResponseField>

<RequestExample>

```bash Básico
curl -X GET "https://worker.thenewscc.com/support/relatorios/agentes" \
  -H "Authorization: Bearer seu_token_jwt"
```

```bash Com Período
curl -X GET "https://worker.thenewscc.com/support/relatorios/agentes?data_inicio=2024-01-01&data_fim=2024-01-31" \
  -H "Authorization: Bearer seu_token_jwt"
```

</RequestExample>
