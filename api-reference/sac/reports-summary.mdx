---
title: "Resumo Rápido"
api: "GET https://worker.thenewscc.com/support/relatorios/resumo"
description: "Retorna resumo dos últimos 30 dias"
---

## Descrição

Retorna um resumo rápido das métricas dos últimos 30 dias sem necessidade de parâmetros. Ideal para dashboards que precisam de uma visão geral rápida. Este endpoint utiliza cache KV com TTL de 2 horas.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token JWT para autenticação
</ParamField>

## Query Parameters

Nenhum parâmetro necessário. O endpoint automaticamente usa os últimos 30 dias.

## Response

<ResponseField name="success" type="boolean" required>
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Dados do resumo">
    <ResponseField name="totalTickets" type="number">
      Total de tickets nos últimos 30 dias
    </ResponseField>

    <ResponseField name="ticketsByStatus" type="object">
      Tickets agrupados por status
    </ResponseField>

    <ResponseField name="ticketsByCategory" type="object">
      Tickets agrupados por categoria principal
    </ResponseField>

    <ResponseField name="resolutionMetrics" type="object">
      Métricas de resolução
    </ResponseField>

    <ResponseField name="ticketTrend" type="array">
      Série temporal diária
    </ResponseField>

    <ResponseField name="periodo" type="object">
      Período analisado (últimos 30 dias)
    </ResponseField>

    <ResponseField name="geradoEm" type="string">
      Data/hora de geração
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="cache" type="object">
  Informações de cache (hit, age, expiresIn)
</ResponseField>

<RequestExample>

```bash
curl -X GET "https://worker.thenewscc.com/support/relatorios/resumo" \
  -H "Authorization: Bearer seu_token_jwt"
```

</RequestExample>
