---
title: "Histórico de Tentativas"
api: "GET https://dashboard-backend.theo-110.workers.dev/quiz/history"
description: "Retorna o histórico de todas as tentativas de quiz do usuário"
---

## Descrição

Lista todas as tentativas de quiz realizadas pelo usuário autenticado, incluindo informações sobre o quiz, data, resultado e pontuação. Os resultados são ordenados por data de tentativa (mais recente primeiro).

## Headers

<ParamField header="Authorization" type="string" required>
  Token JWT do usuário autenticado

  Formato: `Bearer {token}`
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="Dados da resposta">
    <ResponseField name="attempts" type="array" required>
      Lista de tentativas do usuário
      <Expandable title="Estrutura de cada tentativa">
        <ResponseField name="id" type="string">
          ID único da tentativa
        </ResponseField>
        <ResponseField name="quizId" type="string">
          ID do quiz
        </ResponseField>
        <ResponseField name="quizTitle" type="string">
          Título do quiz
        </ResponseField>
        <ResponseField name="quizDate" type="string">
          Data do quiz (formato YYYY-MM-DD)
        </ResponseField>
        <ResponseField name="totalQuestions" type="number">
          Total de perguntas no quiz
        </ResponseField>
        <ResponseField name="correctAnswers" type="number">
          Quantidade de respostas corretas
        </ResponseField>
        <ResponseField name="passed" type="boolean">
          Se o usuário passou no quiz
        </ResponseField>
        <ResponseField name="streakRecovered" type="boolean">
          Se o streak foi recuperado
        </ResponseField>
        <ResponseField name="attemptedAt" type="string">
          Data/hora da tentativa (formato ISO 8601)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
