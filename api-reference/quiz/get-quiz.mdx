---
title: "Obter Quiz"
api: "GET https://sac-waffle-v2.theo-110.workers.dev/quiz/{id}"
description: "Retorna um quiz específico com todas as suas perguntas"
---

## Descrição

Obtém os detalhes de um quiz específico, incluindo todas as perguntas e opções de resposta. As respostas corretas **não são retornadas** neste endpoint para evitar trapaças.

O sistema verifica se o usuário já tentou este quiz. Caso já tenha tentado, retorna erro 403, pois cada usuário tem direito a apenas uma tentativa por quiz.

## Headers

<ParamField header="Authorization" type="string" required>
  Token JWT do usuário autenticado

  Formato: `Bearer {token}`
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único do quiz
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="Dados do quiz">
    <ResponseField name="id" type="string" required>
      ID único do quiz
    </ResponseField>

    <ResponseField name="title" type="string" required>
      Título do quiz
    </ResponseField>

    <ResponseField name="description" type="string" required>
      Descrição do quiz
    </ResponseField>

    <ResponseField name="quizDate" type="string" required>
      Data do quiz (formato YYYY-MM-DD)
    </ResponseField>

    <ResponseField name="minCorrectAnswers" type="number" required>
      Mínimo de acertos para passar (padrão: 6)
    </ResponseField>

    <ResponseField name="questions" type="array" required>
      Lista de perguntas do quiz
      <Expandable title="Estrutura de cada pergunta">
        <ResponseField name="id" type="string">
          ID único da pergunta
        </ResponseField>
        <ResponseField name="question" type="string">
          Texto da pergunta
        </ResponseField>
        <ResponseField name="type" type="string">
          Tipo do campo (sempre "radio" para quiz)
        </ResponseField>
        <ResponseField name="position" type="number">
          Ordem da pergunta (1-10)
        </ResponseField>
        <ResponseField name="options" type="array">
          Array com 4 opções de resposta
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://sac-waffle-v2.theo-110.workers.dev/quiz/quiz_abc123 \
  -H "Authorization: Bearer seu_token_jwt"
```

```javascript JavaScript
const quizId = 'quiz_abc123';

const response = await fetch(`https://sac-waffle-v2.theo-110.workers.dev/quiz/${quizId}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer seu_token_jwt'
  }
});

const data = await response.json();
```

</RequestExample>

<Warning>
  As respostas corretas não são retornadas neste endpoint. Elas são verificadas apenas no momento da submissão via `POST /quiz/submit`.
</Warning>

<Tip>
  O sistema usa letras minúsculas em todas as perguntas e opções, seguindo o estilo do The News.
</Tip>
