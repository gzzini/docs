---
title: "Listar Todos os Quizzes"
api: "GET https://worker.thenewscc.com/quiz/admin/list"
description: "Lista todos os quizzes criados no sistema com paginação"
---

## Descrição

Lista todos os quizzes criados no sistema, ordenados por data de criação (mais recente primeiro). Suporta paginação para facilitar a navegação em grandes volumes de dados. Este endpoint é útil para administração e monitoramento do sistema de quizzes.

<Warning>
  Este é um endpoint administrativo que retorna todos os quizzes do sistema.
</Warning>

## Query Parameters

<ParamField query="page" type="number" default="1">
  Número da página (mínimo: 1)
</ParamField>

<ParamField query="limit" type="number" default="20">
  Itens por página (mínimo: 1, máximo: 100)
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="Dados da resposta">
    <ResponseField name="quizzes" type="array" required>
      Lista de quizzes
      <Expandable title="Estrutura de cada quiz">
        <ResponseField name="id" type="string">
          ID único do quiz
        </ResponseField>
        <ResponseField name="title" type="string">
          Título do quiz
        </ResponseField>
        <ResponseField name="description" type="string">
          Descrição do quiz
        </ResponseField>
        <ResponseField name="quizDate" type="string">
          Data do quiz (formato YYYY-MM-DD)
        </ResponseField>
        <ResponseField name="postId" type="string">
          ID do post do Beehiiv relacionado
        </ResponseField>
        <ResponseField name="minCorrectAnswers" type="number">
          Mínimo de acertos para passar
        </ResponseField>
        <ResponseField name="createdAt" type="string">
          Data/hora de criação (formato ISO 8601)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object" required>
  <Expandable title="Paginação">
    <ResponseField name="total" type="number">
      Total de quizzes no sistema
    </ResponseField>
    <ResponseField name="page" type="number">
      Página atual
    </ResponseField>
    <ResponseField name="limit" type="number">
      Itens por página
    </ResponseField>
    <ResponseField name="totalPages" type="number">
      Total de páginas
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Listar Todos
curl -X GET "https://worker.thenewscc.com/quiz/admin/list"
```

```bash Com Paginação
curl -X GET "https://worker.thenewscc.com/quiz/admin/list?page=1&limit=10"
```

```javascript JavaScript
const response = await fetch('https://worker.thenewscc.com/quiz/admin/list?page=1&limit=20', {
  method: 'GET'
});

const data = await response.json();
```
